# 🔮 Ứng dụng Dự báo Phân lớp Xác suất (PD) với Streamlit

Ứng dụng web này được chuyển đổi tự động từ mã nguồn Python Notebook huấn luyện mô hình **Logistic Regression**. Hệ thống cho phép người dùng tải dữ liệu huấn luyện lên, tinh chỉnh siêu tham số, theo dõi trực quan phân phối biến, kiểm định mô hình bằng biểu đồ ma trận nhầm lẫn trực quan và dự báo tập dữ liệu mới linh hoạt.
